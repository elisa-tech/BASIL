Documentation
=================================

A tool developed to support Software Specification analysis, Software Requirements definition and Test Case mapping against source code or Software Specification.

BASIL is a web application that enable collaboration within multiple users and provide a simplified work item relationships view.
It comes also with a REST web api to simplify the integration in other toolchains.
